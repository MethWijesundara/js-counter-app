# ✨ JavaScript Counter App ✨

A simple counter app built using HTML, CSS, and JavaScript. This Project demonstrates basic DOM manipulation and event handling.

## Features
- Increase the count
- Decrease the count
- Reset the count
- Instant UI updates

## How to Use
1. Open the project in your browser
2. Click "INCREASE" to increment the counter
3. Click "DECREASE" to decrement the counter
4. Click "RESET" to reset the value

## Tech Stack
- HTML
- CSS
- JavaScript
